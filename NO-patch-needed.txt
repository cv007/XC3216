1. Install XC8/XC16/XC32 and preferably PATH enviroment to it. For example i have installed PATH
   XC8 in /opt/microchip/xc8/v2.36/bin/ directory, there is original xclm file.

2. Make on Linux/Windows this source as xclm-hack file and put it in the folder with the xclm file.

    int ii;

    for (ii = 0; ii < argc; ++ii)

        if (!strcmp(argv[ii], "swxc32") ||
            !strcmp(argv[ii], "swxc16") ||

3. Linux(command prompt):
   sudo apt-get update
   sudo apt-get install binfmt-support
   sudo update-binfmts --install xclm-hack /opt/microchip/xc8/v2.36/bin/xclm-hack --magic `(hexdump -e '16/1 "%02x"' -n 64 /opt/microchip/xc8/v2.36/bin/xclm | sed -e 's/../\\\x&/g')`
   sudo service binfmt-support start

   Now the "magic" for xclm-hack should display this format of the original xclm:
